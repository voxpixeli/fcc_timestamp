Timestamp Microservice

usage : provide a date (unix timestamp or a natural language) as parameter

result : get a JSON object with 'unix' and 'natural' dates


examples :

https://silent-playroom.glitch.me/March%208,%202001

 returns {"unix":984009600000,"natural":"March 8, 2001"}

https://silent-playroom.glitch.me/1320837500000

 returns {"unix":1320837500000,"natural":"November 9, 2011"}
